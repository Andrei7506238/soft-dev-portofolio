Fazan Game Bot

C++ | Graph Theory | Algorithms and data structures

Introducing a C++ implementation of a cheat for the classic Romanian word game, Fazan. With this program, players can gain an advantage by having access to a vast array of words at their fingertips. More often than not, the generated word will lock the next player. The program boasts a wide range of options, including toggle for word duplication, the ability to remove words said by opponents, the possibility to play with custom words and more.

Built on a custom directed multigraph data structure and a greedy algorithm, this program guarantees lightning-fast speed even with a massive set of words, including all Scrabble-accepted words in Romanian.

Not only is this software a great tool for gaining a decisive edge in the game, but it can also be used to generate the complete list of words used for blocking. And here's a fun fact: you can link at least 49047 words in a single game of Fazan without closing off options for the next player or repeating a word.

SubInc - Automatic Subtitle Downloader and Softcoder

Python | Web APIs | Windows

A Python-based script that makes adding subtitles to locally downloaded movies and TV shows a breeze. Whether you're looking to improve your language skills or simply prefer watching with subtitles, this script has got you covered. With its ability to recursively batch process directories, the user can quickly and easily add subtitles to their entire media library.

Built using the powerful MKVToolNix toolset and the opensubtitles API, this script ensures there will always be the perfect subtitles at a fingertip.

Not only is it easy to use, but it can also be seamlessly integrated with other tools. Can be added as a contextual menu option for effortless access. Say goodbye to the hassle of manually searching for subtitles, and hello to a more enjoyable viewing experience.

Barbershop Schedule Management System

Java | Swing | SQL

My Java-based schedule management system is a comprehensive desktop application designed to streamline the scheduling process for hairstyle professionals. The application offers flexibility and ease of use, providing an efficient replacement for traditional paper-based agenda methods.

The system supports multiple staff members and languages. It allows efficient management of customers, staff, appointments, and services. The application also offers customizable work schedules and holiday support. Additionally, the system can generate non-blocking warnings for appointments that may overlap, fall outside of the ordinary schedule, and more, providing an added layer of organization and security.

The user interface is designed to be easy to understand, with a clear and intuitive dashboard that provides real-time information about the current status of appointments. The application was developed as a programming tools and environments college project.

OpenGl Fish Aquarium

C++ | OpenGl | 3D

Our OpenGl C++ based game/tech demo showcases our technical and creative proficiency in modern 3D graphical systems. This project showcases a fish aquarium rendered in a modern, aesthetically pleasing room.

The game features advanced graphical techniques, including 3D camera movement, Phong lighting and shadowing, alpha blending of multiple translucide planes, dynamic billboards (e.g. bubbles that are always facing the camera). Additionally, the fish movement is generated using parametric equations, providing a more realistic and dynamic movement.

As the captain of the team, I was primarily responsible for the most challenging aspects of the project, including the management and coordination of the team, as well as the implementation of the most complex features. The project was awarded one of the best demos created in the Modern 3D Graphical Modelling Systems course at the university.
